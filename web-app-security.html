Defense Strategy

Welcome to the 2026 Web Security Protocol. Below are the 5 pillars required for an Information System to maintain the CIA Triad.

01. WAF (Web App Firewall)

Definition: Monitors and filters HTTP traffic. Think of it as a bouncer at the door of your Information System, checking IDs and blocking known troublemakers (SQLi/XSS).

02. Encrypted Data

Definition: Scrambling data at rest. If the physical server hard drive is stolen, the data is useless without the cryptographic key.

03. RBAC

Definition: Role-Based Access Control ensures that an intern cannot access the payroll database. Access is based on the job role, not the individual.

04. MFA

Definition: Something you know (password) + Something you have (phone). This blocks 99% of bulk phishing attacks even if your password is stolen.

05. HTTPS

Definition: Encrypts the "tunnel" between the browser and the server. It prevents "Man-in-the-Middle" attacks where a hacker eavesdrops on a public Wi-Fi.
