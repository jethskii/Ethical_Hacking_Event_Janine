The System Hacking Mindset

System hacking is the art of identifying vulnerabilities in an OS (Windows, Linux, macOS) to move from unauthorized access to complete administrative authority.

The Three Strategic Pillars

1. Infiltration
Bypassing Firewalls & AV via phishing, weak passwords, or unpatched software bugs.
2. Escalation
Climbing the ladder vertically (User → Admin) or horizontally (shifting between same-level accounts).
3. Lateral Movement
Using one 'foot in the door' to expand reach across the entire corporate network using trust relationships.

Business Risk Perspective

In a corporate environment, this isn't just a technical glitch—it's a strategic risk. A single compromised account can lead to:

  • Data Breaches: Exposure of customer PII and financial records.
  • Operational Downtime: Complete paralysis of business workflows.
  • Legal Liability: Massive fines and loss of stakeholder trust.
PRO TIP: Control is rarely taken instantly—it is gained step by step. As defenders, our job is to break those steps.

Breaking the OS Barrier

The Gatekeeper Concept

The Operating System (OS) manages memory, files, and hardware. Breaking the barrier means:

  • Gaining unauthorized entry
  • Escalating privileges (Normal User → Admin)
  • Bypassing built-in security rules
# THE TECHNICAL REALITY: Enumerating unquoted service paths
wmic service get name,displayname,pathname,startmode | findstr /i "Auto" | findstr /i /v "C:\Windows\\" | findstr /i /v """

By finding a service path with a space and no quotes, we exploit a logic flaw where the OS 'guesses' which file to run.

Memory Corruption & BoF

Buffer Overflows (BoF) occur when a program writes more data to a buffer than it can hold, overwriting the Instruction Pointer (EIP).

  • Stack Smashing: Overwriting local variables to hijack execution.
  • NOP Sleds: CPU instructions (\x90) that lead execution to shellcode.
  • The Payoff: Execution of a remote shell directly in system memory.

Active Directory Dominance

In large-scale environments, the Domain Controller (DC) is the ultimate prize. Owning it means owning every identity.

# Kerberoasting: Extracting service hashes for offline cracking
$ invoke-kerberoast -erroraction silentlycontinue

# DCSync: Mimicking a DC to replicate password hashes
$ lsadump::dcsync /domain:corp.local /user:administrator
  • Golden Ticket: Forged TGT using the KRBTGT hash.
  • Silver Ticket: Forged ticket for specific services (SQL/Files).
  • AD CS Abuse: Exploiting Certificate Authorities for impersonation.

Lateral Movement & Graphs

Tools like BloodHound use graph theory to find the "Path of Least Resistance" to the Domain Admin.

  • Pass-the-Hash (PtH): Authenticating with NTLM hashes.
  • Shadow Admins: Backdoors hidden in the AdminSDHolder container.
