System Hacking Foundations

System hacking involves cracking the OS security to gain administrative control. It demonstrates the real-world impact of infrastructure vulnerabilities.

  • Infiltration: Bypassing perimeter defenses (Firewalls/AV).
  • Escalation: Moving from a service account to a System/Admin account.
  • Lateral Movement: Using one compromised machine to infect the entire network.

Breaking the OS Barrier

The Gatekeeper Concept

The Operating System (OS) is the ultimate gatekeeper. It manages memory, files, and hardware. Breaking the barrier means:

  • Gaining unauthorized entry
  • Escalating privileges (Normal User → Admin)
  • Bypassing built-in security rules
# THE TECHNICAL REALITY: Enumerating unquoted service paths
wmic service get name,displayname,pathname,startmode | findstr /i "Auto" | findstr /i /v "C:\Windows\\" | findstr /i /v """

By finding a service path with a space and no quotes, we exploit a logic flaw where the OS 'guesses' which file to run, allowing us to plant a malicious executable.

WHY THIS MATTERS TODAY:
Every business relies on secure OS barriers. Even a small vulnerability—like a missing quote in a file path—can lead to a massive data breach. Awareness is the first line of defense.

Memory Corruption & BoF

Buffer Overflows (BoF) occur when a program writes more data to a buffer than it can hold, spilling into adjacent memory to overwrite the Instruction Pointer (EIP).

  • Stack Smashing: Overwriting local variables to hijack program execution.
  • NOP Sleds: Using \x90 instructions to lead the CPU safely to your shellcode.
  • The Payoff: Execution of a remote shell directly in system memory.

Active Directory Dominance

In large-scale environments, individual machines matter less than the Domain Controller (DC). Owning the DC means owning every identity in the organization.

REAL-WORLD INTEL:
"Attackers often exploit the Service Account Trap. By 'Kerberoasting', we request service tickets and crack them offline to gain a foothold."
# Kerberoasting: Extracting service hashes for offline cracking
$ invoke-kerberoast -erroraction silentlycontinue

# DCSync: Mimicking a DC to replicate password hashes
$ lsadump::dcsync /domain:corp.local /user:administrator
  • Golden Ticket: A forged TGT using the KRBTGT hash.
  • Silver Ticket: A forged ticket for specific services (like SQL or File Shares).
  • AD CS Abuse: Exploiting misconfigured Certificate Authorities.

Lateral Movement & Graphs

Modern hacking isn't a straight line; it's a map. Tools like BloodHound use graph theory to find hidden relationships.

THE ATTACK PATH:
"This is the Path of Least Resistance. An attacker moves from a low-privilege workstation to a high-value server using stolen credentials or session hijacking."
  • Pass-the-Hash (PtH): Authenticating with NTLM hashes.
  • Shadow Admins: Using AdminSDHolder to create a hidden 'backdoor' user.
# Checking for Shadow Admins with GenericAll privileges
Get-DomainObjectAcl -Identity "Domain Admins" | ? {$_.ActiveDirectoryRights -eq "GenericAll"}
